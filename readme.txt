Allows administrators to globally disable comments on their site. Comments can be disabled according to post type.

== Description ==

This plugin allows administrators to globally disable comments on any post types (posts, pages, attachments, etc.) so that these settings cannot be overridden for individual posts.

It requires PHP version 5 or greater.

If you come across any bugs or have suggestions, please contact me at [rayofsolaris.net](http://rayofsolaris.net).

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin settings can be accessed via the 'Settings' menu in the administration area
